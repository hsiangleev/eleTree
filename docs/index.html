eleTree2.0

基于虚拟dom插件snabbdom编写的树组件

起步 →

代码分离

采用webpack，代码结构分离，并支持直接引入，layui模块引入，和npm的方式引入

功能实现

功能包括自定义图标，单选，多选，右键菜单，复制粘贴，懒加载，拖拽，搜索等，并且可以使用pid和children两种结构的数据

性能优化

虚拟dom加后渲染的方式，即初始渲染时，只渲染展开的树节点，上万条数据无压力渲染

兼容性

eleTree2.0版本要求浏览器至少为IE10+
